# BlueScreen Journal v1.0.3

`v1.0.3` focuses on release hardening plus stronger analytics/search workflows for power users and operators.

## What changed

- Added advanced review CLI controls:
  - `bsj review --from YYYY-MM-DD --to YYYY-MM-DD`
  - `bsj review --json`
  - `bsj review --min-count N`
- Added advanced timeline CLI controls:
  - output formats: `--format text|json|csv`
  - aggregate mode: `--summary`
  - metadata filters: `--mood`, `--has-tags`, `--has-people`, `--has-project`
  - weekday filter: `--weekday mon,tue,...`
- Added prompt automation output:
  - `bsj prompts list --json`
  - `bsj prompts pick --json`
- Expanded test coverage for new parser and filter logic.

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.0.3
```
