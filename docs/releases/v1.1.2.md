# BlueScreen Journal v1.1.2

`v1.1.2` is a UX polish release focused on clearer first-run onboarding and stronger regression protection for install/menu flows.

## What changed

- Polished installer completion messaging with a cleaner first-run checklist (launch, type, save, open menus).
- Expanded the post-install interactive menu with:
  - setup guide
  - keyboard/menu cheat sheet
  - `doctor` + PATH repair action
- Improved TUI discoverability messaging:
  - stronger unlock status guidance
  - clearer first menu-open coaching
  - footer/empty-state hints that surface quick-save and menu access
- Expanded QA coverage with new functional UX tests for:
  - menu -> help overlay -> editor roundtrip
  - menu discovery hint transition behavior
  - unlock guidance status flash
- Hardened installer smoke validation to assert first-run flow messaging output.

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.1.2
```
