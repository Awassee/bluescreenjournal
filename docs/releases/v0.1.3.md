# BlueScreen Journal v0.1.3

This release turns the repo and installer into a clearer product surface, not just a code drop.

## Install

Turnkey macOS install:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

## Highlights

- Product-facing GitHub landing page with a clearer value proposition, first-run flow, and daily workflow
- New built-in guides:
  - `bsj guide product`
  - `bsj guide datasheet`
- New public docs:
  - Product guide
  - Datasheet
  - richer setup/distribution docs
- Screenshot and GIF assets for the README and release presentation
- Better nostalgic presentation:
  - menu-driven UI discoverability
  - centered DOS-style `80x25` writing surface
- Release packaging polish:
  - universal macOS artifacts
  - stable asset names for the bootstrap installer
  - packaged doc assets preserved in release bundles and installs

## What bsj gives you right now

- full-screen blue-screen editor for Terminal.app and iTerm2
- encrypted local-first vault
- append-only revisions
- encrypted autosave drafts and crash recovery
- real index and date picker
- in-entry find/replace
- global search with no plaintext disk index
- encrypted folder, S3, and WebDAV sync
- integrity verification
- encrypted backup and restore
