# BlueScreen Journal v0.1.17

This is a stable prebuilt update release focused on updater reliability, clearer in-product controls, and installer freshness.

## Install

Turnkey macOS install:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --version v0.1.17
```

## What improved

- `TOOLS -> Check for Updates` now provides an actionable in-app install option and background updater launch
- `HELP -> About BlueScreen Journal` is now explicit, with version/credit/contact surfaced in-product
- `TOOLS` now includes clearer soundtrack controls:
  - `Soundtrack Source`
  - `Toggle Soundtrack`
- soundtrack auto-start now runs at app launch when a soundtrack source is configured
- installer reruns on already-installed machines now default to a latest-`main` update flow to avoid stale builds
- installer completion output now prints installed `bsj --version` and active `PATH` binary location
