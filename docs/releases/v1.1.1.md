# BlueScreen Journal v1.1.1

`v1.1.1` is a stability and release-surface refresh focused on installer reliability and current-product documentation/screenshots.

## What changed

- Fixed post-install launch behavior to run `bsj` with a PTY wrapper, preventing `Failed to initialize input reader` from installer launch paths.
- Hardened bootstrap release installs so downloaded archives are installed via the current top-level installer logic.
- Removed confusing manual PATH export prompts from installer completion output and improved next-step guidance.
- Refreshed generated screenshot assets to match the current compact/menu-first interface and release marker.
- Updated docs entry points (`README`, docs hub, setup/distribution/start pages, issue template) to the `v1.1.1` release line.

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.1.1
```
