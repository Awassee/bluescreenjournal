# BlueScreen Journal v1.2.1

`v1.2.1` is a release-engineering + QA hardening patch release.

## What changed

- GitHub Actions hardening:
  - upgraded `actions/checkout` to `v6`
  - upgraded `actions/upload-artifact` to `v7`
  - replaced JS release action with `gh release` CLI publishing in workflow
- Added `scripts/manual-smoke-gui-terminals.sh`:
  - installs a published release into a temp prefix
  - validates `bsj --version` and `bsj --help` through Terminal.app and iTerm2
- Added scoped `v1.3.0` execution plan:
  - [`docs/V1_3_PLAN.md`](../V1_3_PLAN.md)
- Updated stable docs/install references to `v1.2.1`.

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.2.1
```
