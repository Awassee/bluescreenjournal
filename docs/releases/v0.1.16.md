# BlueScreen Journal v0.1.16

This is a focused UX polish release for installation and first-run onboarding.

## Install

Turnkey macOS install:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --version v0.1.16
```

## What improved

- installer completion text is now menu-first and avoids implying that CLI parameters are required
- installer PATH messaging is clearer and more user-friendly
- first-run in-app "First 2 Minutes" guidance now emphasizes simple actions and menu discovery
- top-level docs now consistently present CLI checks as optional diagnostics, not required setup steps
