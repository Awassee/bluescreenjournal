# BlueScreen Journal v0.1.15

This is a focused installer reliability patch release.

It fixes the public install experience on fresh machines by making PATH setup persistent across supported shell startup modes.

## Install

Turnkey macOS install:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --version v0.1.15
```

## Fixes

- bundled installer now persists PATH updates automatically when `bsj` is installed outside the active shell PATH
- zsh installs now write PATH guards to both:
  - `~/.zprofile`
  - `~/.zshrc`
- bash installs now write PATH guards to both:
  - `~/.bash_profile`
  - `~/.bashrc`
- installer still prints a one-line `export PATH=...` command for immediate use in the current shell

## Validation improvements

- expanded smoke install checks to verify PATH persistence in isolated homes for:
  - direct bundled install
  - piped bootstrap install with explicit prefix
  - piped bootstrap install with default prefix
  - piped bootstrap install under bash startup files
- smoke archive auto-selection now prefers the newest versioned archive to avoid stale alias artifacts
