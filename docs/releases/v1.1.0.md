# BlueScreen Journal v1.1.0

`v1.1.0` is a feature release focused on faster retrieval workflows, richer timeline controls, optional AI reflection, and updated product docs/screenshots for the current menu-first UI.

## What changed

- Added search power controls:
  - `bsj search --range ...`
  - `bsj search --match-mode all|any|phrase`
  - `bsj search --sort newest|oldest|relevance`
  - `bsj search --hits-per-entry N`
  - `bsj search --summary`
- Added timeline preset management:
  - `bsj timeline --save-preset "Name"`
  - `bsj timeline --preset "Name"`
  - `bsj timeline --list-presets`
  - `bsj timeline --delete-preset "Name"`
- Added timeline calendar grouping output:
  - `bsj timeline --group-by day|week|month`
- Added optional AI reflection surfaces (off by default):
  - `bsj ai summary ...`
  - `bsj ai coach ...`
  - `TOOLS -> AI Summary (Optional)`
  - `TOOLS -> AI Coach Mode (Optional)`
- Added local heuristic fallback for AI summary/coach when remote AI is not configured.
- Refreshed docs surfaces and screenshot assets to reflect the current `v1.1.0` interface.

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.1.0
```
