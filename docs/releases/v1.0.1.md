# BlueScreen Journal v1.0.1

`v1.0.1` is a focused writing-flow release for faster same-day journaling.

## What's new

- Quick-save command in the editor:
  - type `**save**` on its own line
  - press `Enter`
  - BSJ saves immediately and opens the next same-day entry block
- New menu path for the same action:
  - `FILE -> Quick Save + Next`
- Same-day entry scaffolds now appear as:
  - `[ENTRY nn hh:mm]`
  - this keeps multiple entries per day readable without leaving the date
- Updated UI guidance in:
  - empty-state hints
  - footer legend
  - first-run guide
  - help overlay

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.0.1
```
