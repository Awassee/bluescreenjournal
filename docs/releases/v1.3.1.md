# BlueScreen Journal v1.3.1

Release date: 2026-03-19

## Highlights

This release adds a true ten-feature productivity/safety pack in-product, all reachable from `TOOLS -> Insights Center`.

## 10 major features

1. Momentum Snapshot
2. Save Readiness
3. Word Volume
4. Streak Tracker
5. Mood Mix
6. Tag Radar
7. People Radar
8. Project Radar
9. Gap Finder
10. Conflict & Backup Risk

Each module opens as a retro info panel from a single picker so the full flow stays menu-driven and keyboard-only.
