# BlueScreen Journal v0.1.7

This release is a stability-focused product pass that makes the TUI feel more like a complete journaling environment instead of a capable editor with separate power-user surfaces.

It keeps the encrypted vault, sync model, and DOS-era visual contract intact, but adds faster action discovery, better session feedback, and more useful in-product history.

## Install

Turnkey macOS install:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --version v0.1.7
```

## Highlights

- added a searchable command palette with `Ctrl+K`
- added recent entries and favorite dates overlays for faster navigation
- added recent search query recall inside the app
- added built-in writing prompts that insert directly into the editor
- added a configurable daily word goal and live goal progress in the UI
- added a session timer in the header/dashboard
- added persistent sync history with per-run detail
- added richer settings reporting for word goals, favorites, and sync history

## Why this matters

- the app is easier to learn because users can discover commands without memorizing menus
- navigation is faster because recently used dates and starred dates are first-class
- writing momentum improves because prompts, goals, and session feedback are visible while writing
- trust improves because sync history is no longer transient

## User-facing changes

- `Ctrl+K` opens the command palette
- `EDIT` now includes favorite toggling for the current date
- `GO` now includes recent entries and favorite dates
- `SEARCH` now includes recent queries
- `TOOLS` now includes writing prompts and sync history
- `SETUP` now includes a daily word goal setting
- the header now shows goal/session context alongside lock/integrity/save status
