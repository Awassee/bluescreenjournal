# BlueScreen Journal v0.1.8

This release is a large product pass focused on making the TUI feel less like a capable encrypted editor and more like a finished journaling workstation.

It keeps the encrypted vault, append-only history, and DOS-era screen contract intact, but adds a broader in-product command surface, richer writing operations, stronger navigation, and more complete operational feedback.

## Install

Turnkey macOS install:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --version v0.1.8
```

## Highlights

- added a full editor-tools pass with duplicate/delete/move line actions
- added insert helpers for date, time, timestamp, divider, stats, metadata, and blank lines
- added quick export to default text/markdown paths plus export history in the TUI
- added backup policy, integrity details, and encrypted search-cache status overlays
- added favorite-to-favorite navigation and random saved-entry jumps
- added richer calendar navigation across saved days and months with entries
- added favorites-only and conflicts-only filters plus metadata flags in the real index
- added search scope presets for today, current month, and all time
- added new display settings for 12-hour clock, seconds, ruler visibility, and footer legend visibility
- expanded the dashboard, review, quickstart, and help surfaces to explain more of the product inside the app

## Why this matters

- writing feels more complete because common editorial moves no longer require manual text shuffling
- navigation is faster because favorites, random jumps, and saved-date jumps are first-class
- trust improves because backup policy, integrity details, and cache state are visible in-product
- personalization improves because the DOS interface can now be tuned without leaving the TUI

## User-facing changes

- `FILE` now includes quick exports, export history, and backup policy
- `EDIT` now includes line operations and insert helpers
- `SEARCH` now includes range presets and cache status
- `GO` now includes previous/next favorite and random saved entry
- `TOOLS` now includes integrity details and session reset
- `SETUP` now includes clock, seconds, ruler, and footer display settings
- search overlay supports `T`, `M`, `A`, and `C`
- calendar supports saved-date jumps with `[` and `]`, and month-with-entry jumps with `<` and `>`
- index supports favorites-only and conflicts-only filtering with `Shift+F` and `Shift+C`
