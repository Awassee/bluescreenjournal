# BlueScreen Journal v1.2.0

`v1.2.0` is an onboarding + trust visibility release with stronger pre-ship stability gating.

## What changed

- Added `HELP -> First-Run Tour`:
  - menu-accessible guided flow for first 2 minutes
  - explicit daily workflow reminders (write, save, quick-save, menus, go/search/backups)
- Added `TOOLS -> Journal Health`:
  - concise status view for lock state, selected date, `ENTRY NO.`, save state, integrity, backups, conflicts, and sync target
- Added `scripts/stability-gate.sh`:
  - repeat-run critical UX tests with fail-fast behavior
  - integrated into `scripts/qa-gate.sh`
  - enforced in `.github/workflows/release.yml` before packaging
- Refreshed release surfaces to `v1.2.0`:
  - installer examples and doctor hints
  - docs hub/release references
  - README stable links and pinned install examples

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.2.0
```
