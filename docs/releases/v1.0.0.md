# BlueScreen Journal v1.0.0

BlueScreen Journal `v1.0.0` is the first stable release line.

This release locks in the product surface shipped across Milestones 1-10: encrypted local-first journaling, append-only revisions, crash-safe drafts, index/calendar/search flows, sync backends, integrity verification, installer UX, and release packaging.

## Highlights

- Stable `1.0.0` release cut with updated installer/docs pointers.
- MIDI soundtrack playback fixed for macOS by using QuickTime for `.mid/.midi` sources.
- Soundtrack runtime handling improved so playback state reflects real backend state.
- Added test coverage for MIDI source detection and release-doc help-link validation.

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.0.0
```
