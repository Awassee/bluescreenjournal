# BlueScreen Journal v0.1.4

This release is a product-surface maturity pass.

It improves onboarding, support, trust, and public repo usability without changing the core journal model.

## Install

Turnkey macOS install:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

## Highlights

- added a proper docs hub and quickstart path
- added FAQ and product comparison docs
- added support and security policies
- added structured GitHub bug report and feature request templates
- added new built-in guide topics:
  - `bsj guide docs`
  - `bsj guide quickstart`
  - `bsj guide faq`
  - `bsj guide support`
- added changelog tracking at repo root
- improved packaged-install docs so root docs, nested doc trees, and screenshot assets ship together

## Product maturity improvements

- easier first-run navigation for new users
- clearer support and escalation path for operators
- clearer security reporting path
- clearer product-fit guidance for evaluators
- better issue intake for maintainers
