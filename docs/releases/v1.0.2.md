# BlueScreen Journal v1.0.2

`v1.0.2` is a focused soundtrack reliability release for macOS.

## What changed

- Removed QuickTime as the MIDI playback backend.
- Added an internal fallback pipeline for MIDI sources:
  - parse MIDI
  - render to cached WAV
  - play via `afplay`
- This avoids QuickTime compatibility popups for MIDI files that QuickTime refuses to open.

## Why this fixes your issue

Some MIDI files are valid but still rejected by QuickTime/AVFoundation on macOS.  
With this release, BSJ no longer depends on QuickTime for MIDI playback.

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.0.2
```
