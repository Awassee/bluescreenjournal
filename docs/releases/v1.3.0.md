# BlueScreen Journal v1.3.0

`v1.3.0` is the daily-writer confidence release: clearer save flow, stronger in-product guidance, and first-class spellcheck.

## What changed

- Added spellcheck core module (`src/spellcheck.rs`) with:
  - tokenized in-entry misspelling detection
  - suggestion generation
  - common typo correction map
- Added new EDIT actions:
  - `Spellcheck Entry`
  - `Spellcheck Summary`
  - `Next Misspelling`
  - `Previous Misspelling`
  - `Auto-Fix Common Typos`
  - `Add Word At Cursor` (session ignore)
- Added CLI spellcheck:
  - `bsj spellcheck --date YYYY-MM-DD`
  - `bsj spellcheck --from ... --to ...`
  - `bsj spellcheck --range last7 --count-only`
  - `--json` output mode
- Added `FILE -> Save Receipt` overlay for explicit save confirmation.
- Added `HELP -> Daily Flow Coach` menu guidance.
- Added deep archive backward-jump guard (repeat action to confirm).
- Added `docs/V1_3_UX_CHECKLIST.md` and updated `docs/V1_3_PLAN.md` statuses.
- Added regression tests for spellcheck/save-receipt/archive-guard flows.

## Install / Update

Install latest:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash
```

Pin this release:

```bash
curl -fsSL https://raw.githubusercontent.com/Awassee/bluescreenjournal/main/install.sh | bash -s -- --prebuilt --version v1.3.0
```
