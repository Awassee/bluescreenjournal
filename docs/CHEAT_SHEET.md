# First 2 Minutes Cheat Sheet

This is the shortest possible guide to being productive in BlueScreen Journal.

## Write

- start typing immediately
- `F2` saves an encrypted revision
- type `**save**` on its own line, then press `Enter` to save and open the next same-day entry
- `Alt+N` opens the next blank day instead of the next same-day entry

## Find your way around

- `Esc` opens the top menus
- `Ctrl+O` also opens menus if `Esc` feels awkward in your terminal
- `F1` opens help
- `?` also opens help instantly

## Open other entries

- `F3` opens Calendar
- `F7` opens Index
- `F5` opens Search
- older entries are intentionally a little more work to open than today's page

## Core keys

- `F4` find in the current entry
- `F6` replace in the current entry
- `F8` sync
- `F9` closing thought
- `F10` quit
- `F11` reveal codes
- `F12` lock

## Fast movement

- `Alt+,` / `Alt+.` moves day-by-day
- `Alt+Y` or `Alt+0` jumps back to today
- `Alt+[` / `Alt+]` jumps between saved entry dates

## If function keys are awkward

- try `fn` + function key first on Mac laptops
- use `Esc` or `Ctrl+O` for menus
- use the visible menu bar and command strip as the source of truth

## If you only remember three things

1. type immediately
2. `F2` saves
3. `Esc` opens menus
